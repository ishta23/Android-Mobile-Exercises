Prune test

Click/Tap on the shower head or the black circle on the top of the right panel. It will have a red ring round it. This tells the user that they have been in the water too long and are pruney.

Choose a Language

There are 10 langauges at the user can choose from but for the demo there are two: English and French (Francais). Click on either one and see how the time (12hr vs. 24 hr), the on/off button and the choose background changes.

Shower or Tub Use

For this demo there are 2 modes:
Shower - standing in the shower
BathTUb - sitting in the shower
If you click on either one of them, you will see that certain things change and for the bathtub, there are duplicate controls for when you are sitting.

Time, Weather, Timer

This really cool bathroom shows you the time so you can not late. It shows you the weather so you know what clothes to wear when you get out. It also sets a timer for you so you don't hog the water for yourself. The timer display location changes based on if the user is in the bath or shower

Temperature and Scolding Warning

The temperature is simple. It is located on the right panel so it can be seen clearly and changed if need be.
Click on the top arrow of the temperature panel. See that if the temperature is too high, a red warning will show up showing that this temperature can scold you so shut it off and you cannot go any higher.
When the user is lying in a tub, there is a duplicate temperature controls so you can control the water temperature while lying down as well as when you started the bath.

ShowerHead Pressure Preference/Tub Water Level Preference

Based on shower or tub, the icons change.
For shower, there is low shower pressure OR high shower pressure. The user can only select one.
For bath, the user can choose between half full tub or a full tub.

Music

The music is an icon. It displays an album on the other side of the panel so the user can change and do controls and all that.
The music icon is repeated when in sitting/bathtub mode and it is low level for the user to just touch without stretching.

TV/Movie/Media

The tv/movie/media is an icon. It displays an show or Netflix on the other side of the panel so the user can change and change channels or shows.
The music icon is repeated when in sitting/bathtub mode and it is low level for the user to just touch without stretching and it displays right above the bottom tap.

Light

The light is simple feature so click on it so comes on and off. It provides light into the bathtub if there is poor lighting needed.

Background Image

This is an amazing feature. Click on either one (shower and bathtub mode have two different ones) and the new picture is off a waterfall.

Tub Water Level Signal

When the bath tub water level has reached the desired height. There is a selectable screen to see the icon for the demo.
